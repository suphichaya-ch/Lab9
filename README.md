


## 🌩️ WeatherEye: นวัตกรรมการพยากรณ์อากาศด้วย Flutter

**WeatherEye** ไม่ได้เป็นเพียงแค่แอปบอกอุณหภูมิ แต่เป็นระบบที่ถูกออกแบบมาเพื่อความแม่นยำและความลื่นไหล โดยเน้นหลักการสำคัญ 3 ด้าน คือ **Data Integrity**, **State Management**, และ **Modern Design**



### 1. โครงสร้างระบบ (System Architecture)

WeatherEye ใช้โครงสร้างแบบ **Clean Architecture** แบ่งแยกหน้าที่ชัดเจน เพื่อให้ง่ายต่อการบำรุงรักษา:

* **Models:** กำหนดโครงสร้างข้อมูลอากาศที่ซับซ้อน (เช่น `WeatherData`, `DailyForecast`) พร้อมระบบ `fromJson` ที่มีความยืดหยุ่นสูง เพื่อป้องกันแอป Crash จากข้อมูล API ที่ไม่สมบูรณ์
* **Services:** หัวใจหลักในการสื่อสารกับโลกภายนอก โดยเชื่อมต่อกับ **Open-Meteo API** ผ่านโปรโตคอล HTTP เพื่อดึงข้อมูลแบบ Real-time
* **Data Layer:** มีฐานข้อมูลจังหวัดในประเทศไทย (`thaiProvinces`) ที่ระบุพิกัดละติจูดและลองจิจูดอย่างแม่นยำ ทำให้แสดงข้อมูลได้รายพื้นที่



### 2. การจัดการสภาวะแอป (State Management)

เราใช้ **StatefulWidget** ร่วมกับการจัดการแบบ **Reactive** ในจุดที่สำคัญ:

* **Loading States:** มีระบบเช็กสถานะการโหลดข้อมูล (`_isLoading`) เพื่อแสดงผล CircularProgressIndicator ช่วยให้ User รู้ว่าระบบกำลังทำงาน
* **Dynamic Search:** ระบบ Search Engine ภายในแอปที่กรองข้อมูลจังหวัด (Filter) ได้ทันทีขณะพิมพ์ (On-the-fly)
* **Error Handling:** กลไกการดักจับข้อผิดพลาด (Try-Catch) พร้อมแสดงข้อความเตือนเมื่อเกิดปัญหาด้านเครือข่าย ช่วยให้แอปมีความเสถียรสูง (Resilient)



### 3. หลักการออกแบบ UI/UX (Design Principles)

แอปถูกออกแบบภายใต้แนวคิด **"Information at a Glance"** หรือการได้รับข้อมูลครบถ้วนในพริบตาเดียว:

* **Hierarchy:** วางระดับความสำคัญของข้อมูลชัดเจน โดยให้อุณหภูมิปัจจุบันโดดเด่นที่สุด ตามด้วยชื่อจังหวัดและวันที่
* **Visual Representation:** ใช้สัญลักษณ์ (Icons) และข้อความบรรยายภาษาไทยเพื่อให้เข้าใจง่าย
* **Modern Aesthetics:** (ในเวอร์ชันที่ปรับปรุงใหม่) มีการใช้ **Gradients** และ **Glassmorphism** (ความโปร่งใสแบบกระจกฝ้า) เพื่อสร้างมิติและความรู้สึกพรีเมียม



### 4. ฟีเจอร์เด่น (Key Features)

* ✅ **7-Day Forecast:** ไม่ใช่แค่ดูวันนี้ แต่เตรียมตัวล่วงหน้าได้ถึง 1 สัปดาห์
* ✅ **Localized Search:** ค้นหาได้ทั้งภาษาไทยและภาษาอังกฤษ ครอบคลุมทั่วประเทศไทย
* ✅ **Precise Data:** แสดงผลอุณหภูมิสูงสุด-ต่ำสุดอย่างแม่นยำจากพิกัด GPS จริง



**บทสรุป:**
**WeatherEye** คือการผสมผสานระหว่างการเขียนโปรแกรมเชิงโครงสร้างที่แข็งแรง (Solid Code) เข้ากับการออกแบบที่เน้นผู้ใช้เป็นศูนย์กลาง (User-Centric Design) ทำให้แอปนี้เป็นเครื่องมือที่เชื่อถือได้ในทุกสภาพอากาศ
